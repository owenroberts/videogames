Blender to Unity Asset Pipeline

Video Games > Week 3

Preparing the blender model

  • Unwrap UV map
  • Object Mode
  • Export FBX
    • FBX 7.4 binary
    • Selected objects (to ignore Camera, Lamp etc)
    • Only select the Mesh for now
    • Scale leave at 1.00, maybe change later
  • Save .blend files in separate Blenders folder

Importing in Unity

  • Set up Unity (Create account etc...)
  • Create Unity project
  • Put .fbx file in Models folder inside Assets
  • Drag .fbx from the Models folder to the Hierarchy
  • Check size, rotation
  • Create a Materials folder and add default material

Textures in Blender

  • External images
  • UVs
  • Export UV layout
  • Painting textures (save image)
